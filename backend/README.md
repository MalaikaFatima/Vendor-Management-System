# Vendor Management System

RESTful API built with Laravel 12 for managing vendors, quotations, and quote submissions.

## Features

-   Authentication using Laravel Sanctum
-   Vendor CRUD Operations
-   Vendor Search & Filtering
-   Quotation Management
-   Quote Submission & Comparison
-   Cheapest Quote Detection
-   Dashboard Statistics
-   Recent Activities
-   Pagination & Validation

## Tech Stack

-   Laravel 12
-   PHP
-   MySQL
-   Laravel Sanctum

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
